ArchipelagoEngine

Standard spatial contiguity models often leave significant portions of island nations mathematically isolated. In the Philippine context, standard Queen logic leaves 16 provinces (approx. 20%) orphaned.

100% Connectivity Achieved: ArchipelagoEngine bridges maritime networks using optimized KNN logic.

Figure 1: Standard Queen Logic (Left) vs. ArchipelagoEngine k=5 (Right)

Quick Start

install.packages("ArchipelagoEngine")
library(ArchipelagoEngine)
weights <- build_archipelago_weight(raw_data, k = 5)
